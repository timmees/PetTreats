# PetTreats
Full Stack Web Dev Project from Tim Mees and Paul Jauert.


Dozent: Prof. Dr. Alexander Eck

Projekt von Tim Mees und Paul Jauert


Idee: Web-App, welches ein Leckerlis-Abo für Haustiere anbietet

Problem: Haustierbesitzer vergessen oft, regelmäßig Futter oder Snacks zu kaufen.

Lösung/Umsetzung: Ein Mini-Webshop mit wählbarem Monatsabo für Snacks je nach Tierart.
Login-und Userbereich, Bestellverwaltung
Produktanzeige mit Bildern und Beschreibung 

Target User: Tierbesitzer, insbesondere Besitzer von Hunden, Katzen, Hamster etc.

Technische Umsetzung mit Flask, SQLite, JSON, Jinja, Python
